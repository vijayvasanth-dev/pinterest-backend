Pinterest Clone – MERN Stack
A full-stack Pinterest-inspired web application built with the MERN stack.
Live Demo: https://pinterest-cb2669.netlify.app


Frontend
Hosted on: Netlify
Repository: GitHub – https://github.com/vijayvasanth-dev/pinterest-frontend
Features

Responsive homepage with pin grid layout
Login and Register screens
Create, Edit, and Delete pins
Pin detail page
Like pins dynamically
User profile page
Protected routes (only logged-in users can access)

Tech Stack

React.js
React Router DOM
Axios


Backend
Hosted on: Render
respository:Github - https://github.com/vijayvasanth-dev/pinterest-backend
Features

Authentication and Authorization using JWT and HTTP-only Cookies
REST API for pins and users
Image upload via Cloudinary
Dynamic like count saved to database
User profile management
Database integration with MongoDB Atlas

Tech Stack

Node.js
Express.js
MongoDB Atlas + Mongoose
JWT
Cloudinary
Multer
